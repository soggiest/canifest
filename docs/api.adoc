= Api
The Canifest API provides a single entrypoint to all Canifest functions

== Version
Version: -100

== Get Methods

=== instructions
Lists available instuctions
-----------
GET /api/v-100/instructions
-----------
Response:
-----------
[
    "FROM",
    "LABEL",
    "COPY",
    "RUN",
    "CMD",
    "ENV",
    "ARG",
    "EXPOSE",
    "USER",
    "VOLUMES",
    "WORKDIR",
    "ONBUILD",
    "ENTRYPOINT",
    "STOPSIGNAL"
]
-----------
Example:
-----------
curl http://canifest.example.com/api/v-100/list
-----------
